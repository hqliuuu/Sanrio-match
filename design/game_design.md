# 🎮 三丽鸥梦幻消消乐 - 游戏设计文档

## 1. 游戏概述

### 1.1 核心玩法
- **类型**：经典三消（Match-3）游戏
- **平台**：微信小程序
- **目标用户**：女性玩家为主，年龄段6-35岁
- **游戏时长**：单局3-5分钟

### 1.2 游戏特色
- 三丽鸥官方正版授权角色
- 丰富的故事剧情模式
- 甜美可爱的美术风格
- 60+精心设计的关卡
- 角色收集和养成系统

## 2. 核心机制

### 2.1 消除规则
1. **基础匹配**：3个相同颜色元素横向或纵向连成一线即可消除
2. **特殊元素**：
   - 4连：生成横向/纵向条纹糖果（消除整行/整列）
   - L/T型5连：生成包装糖果（3x3爆炸）
   - 5连：生成彩虹糖（消除所有同色元素）
   - 特殊组合：两个特殊元素交换产生更强效果

### 2.2 关卡目标
| 目标类型 | 说明 |
|---------|------|
| 得分模式 | 达到指定分数即可过关 |
| 收集模式 | 收集指定数量的特定元素 |
| 清除模式 | 清除指定数量的障碍物 |
| 下落模式 | 让指定数量的目标元素落到底部 |
| Boss模式 | 对Boss造成指定伤害 |

### 2.3 障碍物系统
- **冰块**：需要消除旁边元素1次打破
- **巧克力**：每回合会蔓延一格
- **牢笼**：需要匹配内部元素才能打开
- **果冻**：需要消除2次才能清除
- **石板**：无法消除，但可以下落

## 3. 经济系统

### 3.1 货币
- **金币**：基础货币，用于购买道具、复活
- **星星**：高级货币，用于解锁章节、购买皮肤

### 3.2 道具
| 道具 | 功能 | 价格（金币） |
|-----|------|-------------|
| 锤子 | 消除任意一个元素 | 500 |
| 炸弹 | 消除3x3区域 | 800 |
| 彩虹 | 消除所有同色元素 | 1000 |
| 洗牌 | 重新排列棋盘 | 600 |
| 额外步数 | +5步 | 300 |
| 额外时间 | +30秒 | 400 |

## 4. 社交功能
- 好友排行榜
- 送心系统（赠送体力）
- 关卡求助
- 成就分享

## 5. 关卡设计

### 5.1 难度曲线
- 1-10关：简单教学，引入基本机制
- 11-30关：中等难度，引入障碍物
- 31-50关：困难，需要策略规划
- 51-60关：专家难度，多种机制组合
