## 🎨 七、AI绘画批量生成方案

#### 步骤1: 准备提示词列表
已在上文提供每个素材的Midjourney提示词，复制即可使用。

#### 步骤2: 批量生成脚本
```bash
# 使用 Midjourney 的 /imagine 命令逐个生成
# 或使用第三方批量生成工具
```

### 使用 Stable Diffusion 批量生成

#### 推荐模型
- **Model**: Counterfeit-V3.0 或 MeinaMix
- **LoRA**: Sanrio style LoRA
- **VAE**: pastel-waifu-diffusion

#### 批量生成脚本 (Python)
```python
import requests
import json

# Stable Diffusion API 批量生成
assets_to_generate = [
    {
        "filename": "piece_strawberry.png",
        "prompt": "masterpiece, best quality, cute kawaii strawberry...",
        "negative_prompt": "low quality, bad anatomy...",
        "width": 100,
        "height": 100
    },
    # ... 更多素材
]

for asset in assets_to_generate:
    # 调用 SD API
    pass
```

### 使用 DALL-E 3 批量生成

#### 推荐设置
- **Quality**: hd
- **Style**: vivid
- **Size**: 1024x1024 (后期裁剪)

## 📋 八、素材命名规范

### 命名格式
```
[类型]_[名称]_[状态].png

示例:
- piece_strawberry.png (消除元素-草莓)
- piece_strawberry_glow.png (消除元素-草莓发光版)
- bg_chapter1.png (背景-第一章)
- btn_start.png (按钮-开始)
- btn_start_pressed.png (按钮-开始按下状态)
- icon_star.png (图标-星星)
- effect_match_01.png (特效-消除帧1)
```

### 文件夹结构
```
assets/images/
├── pieces/
│   ├── normal/          # 普通元素
│   ├── special/         # 特殊元素
│   └── effects/         # 元素特效
├── characters/
│   ├── hello_kitty/
│   ├── my_melody/
│   └── ...
├── ui/
│   ├── buttons/         # 按钮
│   ├── icons/           # 图标
│   ├── frames/          # 边框
│   └── backgrounds/     # 背景
├── items/               # 道具
└── effects/             # 游戏特效
```
