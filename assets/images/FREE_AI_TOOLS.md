## 推荐免费工具（无需订阅）

### 1. 🥇 Bing Image Creator (最推荐)
**网址**: https://www.bing.com/create
**费用**: 完全免费
**优点**:
- 基于 DALL-E 3，质量高
- 每天可生成100+张图片
- 无需付费，微软账号即可
- 支持中文提示词

**使用方法**:
1. 登录微软账号
2. 输入提示词（支持中文）
3. 点击生成，等待1-2分钟
4. 直接下载高清图

**适合生成**:
- ✅ 消除元素（水果、糖果）

**使用方法**:

3. 选择模型："DreamShaper" 或 "Anime"
4. 输入提示词生成

**适合生成**:

## 🚀 快速开始指南

### 第一步：注册账号（5分钟）

1. **微软账号**（用于Bing）
   - 访问 https://signup.live.com
   - 注册免费账号

2. **Leonardo.AI**
   - 访问 https://leonardo.ai
   - 用谷歌/微软账号登录

### 第二步：开始生成

**示例 - 生成草莓消除元素：**

**Bing Image Creator 提示词：**
```
A cute kawaii strawberry game piece icon, Sanrio style, 
round and plump, shiny gloss effect, pastel red color, 
simple and clean design, white background, 3D render
```

**Leonardo.AI 设置：**
- Model: DreamShaper v7
- Width: 512
- Height: 512
- Guidance Scale: 7

### 第三步：批量处理

使用免费工具处理生成的图片：

1. **Remove.bg**（免费版）- 去除背景
2. **Photopea**（免费在线PS）- 调整大小
3. **TinyPNG**（免费）- 压缩图片

## ⚠️ 注意事项

1. **版权**：免费工具生成的图片通常可商用，但建议查看各平台最新条款
2. **一致性**：AI生成的系列图片可能风格不完全一致，建议使用相同种子值
3. **后期处理**：生成的图片通常需要裁剪、去背景、调整大小
