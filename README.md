# 🎀 三丽鸥梦幻消消乐 🎀

一个以三丽鸥（Sanrio）为主题的微信小游戏消消乐。

## ✨ 功能特性

### 🎮 核心玩法
- **经典三消玩法**：简单易上手，关卡丰富
- **特殊糖果系统**：条纹糖果、彩色炸弹等特殊元素
- **连锁反应**：消除触发连锁，获得更多分数

### 🧊 道具系统（新增）
| 道具 | 图标 | 功能 | 解锁关卡 |
|------|------|------|----------|
| +5步 | +5 | 增加5步移动次数 | 第1关 |
| 锤子 | 🔨 | 消除任意一个元素 | 第2关 |
| 刷新 | 🔄 | 重新排列所有元素 | 第3关 |
| 冰冻 | ❄️ | 冻结时间5秒 | 第5关 |
| 炸弹 | 💣 | 消除3x3区域 | 第7关 |
| 彩虹 | 🌈 | 消除所有同色元素 | 第10关(Boss) |

### 🧱 障碍物系统（新增）
| 障碍物 | 图标 | 特性 |
|--------|------|------|
| 冰块 | 🧊 | 消除一次后解除，不阻止移动 |
| 岩石 | 🪨 | 需要消除周围元素来破坏 |
| 笼子 | 🔒 | 锁住元素，不能直接移动 |
| 巧克力 | 🍫 | 每3步会蔓延到相邻格子 |
| 奶油 | 🥛 | 双层，需要消除两次 |
| 果冻 | 🍮 | 关卡目标，必须消除才能过关 |

### 📖 故事模式
- **第一章**：Hello Kitty的甜品派对（10关）
- **第二章**：美乐蒂的音乐盒（10关）
- 每关都有独特的前后剧情对话
- 三个角色：Hello Kitty、美乐蒂、大耳狗

### 🎨 游戏界面
- 精美的三丽鸥风格UI
- 章节主题背景
- 角色立绘和头像
- 暂停菜单功能

## 🎮 操作说明

### 游戏内操作
- **点击元素**：选中/取消选中
- **点击相邻元素**：交换位置
- **点击道具**：使用道具
- **点击棋盘（使用需要目标的道具后）**：选择目标位置
- **点击暂停按钮**：暂停游戏

### 暂停菜单
- **继续游戏**：回到游戏
- **重新开始**：重新开始当前关卡
- **返回首页**：回到主菜单

## 🔧 技术架构

### 模块职责
- **main.js**: 游戏状态机、场景切换、渲染分发
- **game.js**: 游戏核心循环、触摸处理、道具交互
- **board.js**: 棋盘状态、元素位置、渲染
- **powerups.js**: 道具效果实现、冷却时间管理
- **obstacles.js**: 障碍物状态、破坏逻辑
- **level.js**: 关卡数据解析、道具解锁逻辑

## 🚀 开发说明

1. 使用微信开发者工具打开本项目
2. 在 `project.config.json` 中配置你的 AppID
3. 开始开发吧！

### 添加新道具
1. 在 `powerups.js` 中添加道具类型和配置
2. 在 `POWERUP_CONFIG` 中定义道具属性
3. 实现道具效果方法（如 `useXXX()`）
4. 在 `level.js` 中配置解锁关卡

### 添加新障碍物
1. 在 `obstacles.js` 中添加障碍物类型
2. 在 `OBSTACLE_CONFIG` 中定义属性
3. 在关卡数据中配置 `obstacles` 数组
